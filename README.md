# British Airways: Data Science & Strategic Analytics Portfolio

This repository contains industry-focused data solutions developed for British Airways, covering both digital customer acquisition and ground operations.

---

## 🛋️ Project 1: Heathrow T3 Lounge Demand Forecasting
**Business Case:** Optimize premium lounge capacity and staffing at Heathrow Terminal 3.

* **Logic:** Developed a strategic model to categorize 1,000+ passengers based on cabin class and Executive Club status.
* **Action:** Built a reusable automated lookup tool in Excel to forecast peak occupancy hours.
* **Outcome:** Provided data-backed justifications for capacity adjustments during peak flight "banks."
* **Tools:** Advanced Excel (XLOOKUP, Nested Logic, Demand Modeling).

---

## ✈️ Project 2: Customer Booking Predictive Modeling
**Business Case:** Predict high-intent customers to increase flight booking conversions.

* **Action:** Cleaned and analyzed customer clickstream data to identify patterns in purchase lead times and length of stay.
* **Modeling:** Trained a **Random Forest Classifier** (Scikit-learn) to predict booking probability.
* **Key Insight:** Discovered that 'purchase lead time' is a primary driver for conversion, allowing for targeted marketing strategies.
* **Tools:** Python (Pandas, Scikit-learn), Matplotlib, Jupyter Notebooks.
